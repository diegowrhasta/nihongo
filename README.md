# Nihongo

こんにちは、ディエゴです。日本に住んでいるウェブソフトウェアエンジニア！

スペイン語を話せる人として、英語ペラペラになっちゃったけど、実はロマンに影響受けられた
言語から、アジアの言語を勉強することはめちゃくちゃ大変だよ。本当に大変。

文法とか、言語とか、発音とか、全部は全然違う。しかも、考え方でも全然違う、もし、
ちゃんと話せたかったら、一生懸命に日本語だけでなく、日本の文化について勉強しなきゃ。

僕は、もう4年前から勉強し始めたけど、いまでも下手くそである。今から、このレポで
面白い日本語を勉強しながら見つけたことについて書くの。よろしくお願いします。

## ANKI

Youtubeを見ると、日本語を勉強する人は皆、「ANKIはぜったい」っていつも言うけど、
個人的には、これは詐欺だと思う。丸ではなくて、SRS、それで、暗記だけ十分だったら、
日本人の皆は英語ペラペラに話せること。それは事実ですか？全然ないですよね。

僕はAnkiで始めたけど、勉強すればするほど、手がつけられなくなるよ、絶対。ある時
2000枚のカードをレビューしないと駄目だった。だからこそ、知ってる。1000時間で愚かな
勉強したら、それは意味がない時間だと思う。意味が無し。だから、よく調べたら、ちゃんと
大事なことに集中したら、それの方が良い。

だから、今の助言は：

ANKIは1日辺り20枚じゃない！3−7枚ならでいい。しかも、20分間では絶対。20分間なら
明日のレビュウ量は勉強できる量である。嫌なら、よく気づいて、それはあんたの体、
あんたの脳の知らせ。「これは駄目、止め！」。試験のためにではなく、人生のためには
言語を勉強することである。それだけ！
